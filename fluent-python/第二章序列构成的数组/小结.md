# 内置的序列类型
1.按照存放类型划分
- 容器序列：list, tuple, collections.deque。存放多种类型
- 扁平序列：str、 bytes、 bytearray、 memoryview 和 array.array。一种类型

2.按是否能被修改：
- 可变类型：list、 bytearray、 array.array、 collections.deque 和 memoryview
- 不可变类型：str, tuple, bytes
